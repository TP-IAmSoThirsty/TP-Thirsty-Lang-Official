Thirsty-Lang Documentation

Thirsty-Lang (Universal Thirsty Family — UTF) is a governance-first programming language family designed to make Red Hat and Black Hat attack patterns structurally harder through water-metaphor syntax.

Module API Reference

  • thirsty_lang Tier 1 — Core language: lexer, parser, AST, type system, interpreter, CLI, formatter, module system
  • thirst_of_gods Tier 2 — Deity contract validation for OOP, async, error handling
  • tarl Tier 3 — T.A.R.L. policy-as-code engine with default-DENY
  • shadow_thirst Tier 4 — Mutation analysis and invariant verification
  • tscg Tier 5 — Symbolic constraint grammar with 9 core symbols
  • tscg_b Tier 6 — Binary frame protocol with CRC32/SHA-256

Language Specification

  • LANGUAGE_SPEC.md — Human-readable language reference covering all 6 tiers
  • GRAMMAR.md — Formal BNF/EBNF grammar for Thirsty-Lang Tier 1
